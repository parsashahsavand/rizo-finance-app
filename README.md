## ✨ Features

- **Quick Transaction Logging** — Add a new transaction in just 3 steps
- **Offline AI Analysis** — Smart financial insights with no internet or server required
- **Challenge System** — Set personal financial goals and earn stars for completing them
- **Financial Reports** — Clear overview of income, expenses, and savings

## 🛠 Tech Stack

| | |
|---|---|
| Language | Kotlin |
| UI | Jetpack Compose |
| Architecture | MVVM + Clean Architecture |
| Database | Room |
| DI | Hilt |
| Storage | DataStore |

## 🎯 About

Rizo is built for people who want simple, no-nonsense control over their daily finances — no bank connection, no complex setup, just clarity.
